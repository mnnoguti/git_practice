Hello Git and GitHub

git init //para tornar o diretório atual em um repositório git

git diff filename //exibe diferença do arquivo commited com o atual

git log //exibe todos os commit log

git status //para verificar se tem arquivos para atualização

git add filename //adiciona arquivo para Staging Area

git commit -m "Commit OK" //para commit
